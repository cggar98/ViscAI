# ViscAI library
==============================================================

**ViscAI** is a Streamlit GUI to execute rheology simulations with **BoB (Branch-on-Branch)**, run parallel parameter sweeps, and produce structured datasets with **SQLite** ready for analysis and model fitting with **pyRheo**.
